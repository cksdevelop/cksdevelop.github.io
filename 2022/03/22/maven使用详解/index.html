maven使用详解

Created2022-03-22|Updated2022-04-20
|Post View:

符号说明

IDEA使用Maven配置

  • File | Settings | Build, Execution, Deployment | Build Tools | Maven

    Maven home directory:[ 没什么好解释的,相当于一个软件安装在本地的目录,可以安装不同的maven版本 ]

    User settings file:[ 使用maven要用的setting设置,与远程仓库的的连接啊,什么等等配置 ]

    Local repository:[暂时存储在本地的项目依赖仓库,就是为了方便加载 ]

项目结合Maven插件工具使用

  • Maven | [ {项目名称} ] | [ Lifecycle | Plugins | Dependencies ]

  • Maven | [ {项目名称} ] | [ Lifecycle ] | [ deploy ]

    要学会使用 Maven插件栏的 刷新按钮 [一个双箭头组成的圆],可解决很多软件问题错误
