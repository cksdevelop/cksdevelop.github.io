写一篇的基本步骤：

（这段文字的意义在于：如果我很久不写博客了，那他就是我的抗遗忘大使了，既然是抗遗忘那就得有置顶功能****和标签功能**）

​ ①打开hexo的根目录blog：通过鼠标右键选择

​ ②在黑窗口中输入 hexo new <文件名>

​ ③在快速访问里打开source–>_posts–><文件名>.md文件

​ ④检查该md的偏好设置和格式下的图像–>设置图像根目录为source文件夹

​ ⑤开始……

maupassant的置顶功能实现

查阅文档说明：

1、tags标签功能的实现: 的使用

​ 我说怎么老报错了，冒号“：”后面要有个空格，再写tags参数。其他参数前都是要有空格的…

2、wordcount文章字数统计功能实现：

使用hexo-wordcount进行文章字数统计

​ ①通过cnpm i --save hexo-wordcount命令安装hexo-wordcount

​ ②修改主题文件下的_config.yml里的wordcount参数值为 true

​ ③黑窗口输入 hexo clean && hexo g清理缓存以及本地部署静态html文件

​ ④黑窗 hexo s本地localhost:4000访问 本地静态html文件

​ ⑤localhost:4000访问没毛病，就hexo d同步到主页
